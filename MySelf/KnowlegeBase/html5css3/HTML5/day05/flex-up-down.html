一个或最后一个基准集的对齐基线与基线共享组中所有框的共享的第一个或最后一个基准集中的相应基线对齐。 回退对准first baseline是start，一个last baseline是end。 space-between 物品沿着主轴均匀分布在对准容器内。每对相邻物品之间的间距相同。第一个项目与主启动边缘齐平，最后一个项子项的项目，此值被视为end。 center 这些物品沿着主轴线朝向对准容器的中心彼此齐平地填充。 left 物品彼此齐平地值。 正式语法 flex-start | flex-end | 中心| 空格之间| 空间| 空间均匀 例编辑 CSS内容 #container { display: flex; justify-content: space-between; /* Can be changed in the live sample */ } #container > div { width: 100px; height: 100px; background: linear-gradient(-45deg, #788cff, #b4c8ff); } 结果
